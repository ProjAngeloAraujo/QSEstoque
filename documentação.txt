Clonar o repositorio;
Baixar o py 3.12.10;
Ativar a venv:
    py -3.12 -m venv venv;
    venv\Scripts\activate (bash);
Baixar as bibliotecas:
    pip install django psycopg[binary];
    python.exe -m pip install --upgrade pip;
Criar o projeto:
    django-admin startproject QSEstoque . ;
    python manage.py startapp APPEstoque (criar o APP);
    Criar a pasta de templates, static e urls.py do APP;
    Ambiente Virtual;

    Depois de instalar o PostgreSQL, copie o caminho dele (**GERALMENTE** C:\Program Files\PostgreSQL\15\bin);

    CREATE DATABASE qsestoque_db;

    ALTER USER angelousuario WITH SUPERUSER;
